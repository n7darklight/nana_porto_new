Batique: Batik Pattern Recognition

A Mobile App for Indonesian Batik Pattern Recognition

Project Overview

Batique was developed as the capstone project for the Bangkit Academy 2021 program, led by Google, Gojek, Tokopedia, & Traveloka. As the machine learning lead, I was responsible for the core technology that powers the application.

The primary goal of the project was to create an educational tool that helps users identify and learn about the rich heritage of Indonesian Batik. By simply taking a photo, the app recognizes the pattern and provides historical and cultural information. This involved building a robust computer vision model capable of classifying various intricate Batik designs with high accuracy.

Key Features

  • Real-time Pattern Recognition: Utilizes the device's camera for instant identification of Batik patterns.
  • High-Accuracy Classification: Employs a deep learning model fine-tuned on a custom dataset of Indonesian Batik.
  • Educational Content: Provides users with the name, origin, and meaning behind each recognized pattern.
  • User-Friendly Interface: Designed for a seamless and intuitive mobile experience.

Tech Stack

  • TensorFlow & Keras
  • Mobile App (Android)
  • Computer Vision
  • Python
